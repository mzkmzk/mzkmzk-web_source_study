##1. Button

![Button效果](QQ20151216-0.png)

主要效果有:

1. 间距设置
2. `hover`时按钮有阴影
        @shadow: inset 0 1px 0 rgba(255,255,255,.15), 0 1px 1px rgba(0,0,0,.075);
3. 圆角设置
4. 点击的时候,阴影加深
5. 点击了按钮之后,边框出现蓝色,失去焦点后才消失
